# 📊 Customer Behavior Analysis Dashboard

## 📌 Project Overview

The Customer Behavior Analysis Dashboard is an end-to-end data analytics project that analyzes customer purchasing behavior, demographics, and sales performance. This project demonstrates the complete data analytics workflow by integrating Python, MySQL, SQL, and Power BI to transform raw data into meaningful business insights.

The project begins with Exploratory Data Analysis (EDA) using Python, where the dataset is cleaned, explored, and analyzed to identify trends and patterns. The cleaned dataset is then imported into MySQL Workbench, where SQL queries are used to extract business insights. Finally, the processed data is connected to Power BI to create an interactive dashboard that helps users analyze customer behavior through dynamic visualizations and Key Performance Indicators (KPIs).

---

# 🎯 Business Objective

The objective of this project is to help businesses understand customer behavior by analyzing customer demographics, subscription status, purchasing patterns, product performance, and revenue trends. The dashboard enables business users to make informed decisions using interactive reports and visual analytics.
